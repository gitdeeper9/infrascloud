# INFRAS-CLOUD Notebooks

This directory contains Jupyter notebooks demonstrating various aspects of the INFRAS-CLOUD framework.

## Notebook Index

1. **01_quickstart.ipynb** - Quick start guide and basic usage
2. **02_aisi_validation.ipynb** - AISI validation on 1,847 events
3. **03_case_study_irma_2017.ipynb** - Hurricane Irma (2017) case study
4. **04_case_study_hunga_tonga_2022.ipynb** - Hunga Tonga eruption (2022)
5. **05_case_study_tornado_outbreak_2011.ipynb** - 2011 Super Outbreak
6. **06_microbarom_ocean_inversion.ipynb** - Ocean wave inversion
7. **07_stratospheric_ducting_profiling.ipynb** - Stratospheric wind profiling
8. **08_climate_change_projections.ipynb** - Climate change implications

## Usage

```bash
jupyter notebook
```

Select the desired notebook from the browser interface.
