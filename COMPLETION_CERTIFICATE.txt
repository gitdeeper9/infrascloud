╔════════════════════════════════════════════════════════════════════════════╗
║                         PROJECT DESCRIPTION                                ║
╠════════════════════════════════════════════════════════════════════════════╣
║                                                                            ║
║   INFRAS-CLOUD (Infrasonic Network for Real-time Atmospheric               ║
║   Signatures --- Continuous Low-frequency Observatory for Unresolved       ║
║   Dynamics) is a physics-grounded eight-parameter framework for the        ║
║   continuous, real-time acoustic monitoring of severe atmospheric          ║
║   phenomena through analysis of infrasonic wave fields in the frequency    ║
║   band 0.001--20 Hz.                                                        ║
║                                                                            ║
╚════════════════════════════════════════════════════════════════════════════╝


╔════════════════════════════════════════════════════════════════════════════╗
║                        THE EIGHT PARAMETERS                                 ║
╠════════════════════════════════════════════════════════════════════════════╣
║                                                                            ║
║   ┌──────┬─────────────────────────┬──────────────┬───────────────┐      ║
║   │  #   │ Parameter               │ Symbol       │ Weight (w_i)  │      ║
║   ├──────┼─────────────────────────┼──────────────┼───────────────┤      ║
║   │  1   │ Microbarom Amplitude    │ P_ub         │     0.18      │      ║
║   │  2   │ Stratospheric Ducting   │ D_str        │     0.14      │      ║
║   │  3   │ Spectral Peak Frequency │ f_p          │     0.21      │      ║
║   │  4   │ Azimuthal Arrival Angle │ θ            │     0.15      │      ║
║   │  5   │ Phase Velocity          │ v_ph         │     0.12      │      ║
║   │  6   │ Atmospheric Absorption  │ α_air        │     0.07      │      ║
║   │  7   │ Inter-station Coherence │ γ²           │     0.08      │      ║
║   │  8   │ Signal-to-Noise Ratio   │ SNR          │     0.05      │      ║
║   └──────┴─────────────────────────┴──────────────┴───────────────┘      ║
